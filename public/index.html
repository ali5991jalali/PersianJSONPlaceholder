Introduction

This is a free online REST API that returns Persian data (RTL data). You can use whenever you need some fake data.

HTTP methods

HTTP methods, Include GET, POST, PUT, PATCH and DELETE.

Persian data

Returns Persian (Right to Left) fake data as JSON file format that.

With love

It's free and made with love for Iranian developers (and who needs persian fake data for ajax requests)

Example

You can use this service with GraphQL, visit here to see examples about it.

And you're able to run this code in a console or from any site:

fetch('https://jsonplaceholder.ir/todos/1')
        .then(response => response.json())
        .then(json => console.log(json))

        /* Tip: you can use https or
        
 http for make requests :) */
